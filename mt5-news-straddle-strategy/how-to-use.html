Como Usar el EA v2.01

Guia para operar noticias con straddle automatico, trailing stop y gestion de TP/SL

Novedades v2.01
  • Market Orders (v2.00): El EA ya no usa ordenes pendientes (Buy/Sell Stop). Ahora monitorea el precio y ejecuta ordenes de mercado cuando el precio cruza los niveles calculados. Mas robusto y sin rechazos por tipo de orden.
  • Trailing Stop (v2.01): Trailing stop clasico configurable via InpTrailingStopPips. Se activa solo cuando la posicion alcanza breakeven o mejor. Protege ganancias automaticamente.
  • Take Profit On/Off (v2.01): Toggle InpEnableTP permite activar o desactivar el TP. Con TP off + trailing, la posicion puede correr indefinidamente.
  • Panel Minimizable (v1.16): Boton "-" / "+" en la esquina superior derecha del panel para minimizar/restaurar.
Inicio Rapido
1

Carga el EA y Configura

Arrastra NewsStraddleEA al grafico. Configura InpNewsTime con la hora exacta del evento (hora del servidor MT5), InpEntryDistPips, InpSLPips, InpTPPips, y InpTrailingStopPips.

2

El EA se arma automaticamente

Al cargar, el EA entra en ARMED y muestra el contador T-MM:SS. A T-10 segundos (configurable via InpLeadTimeSeconds) calcula los niveles de entrada (lineas verde y roja en el chart) y pasa a MONITORING.

3

Monitoreo de precio y entrada por market order

El EA vigila el precio en cada tick. Cuando Ask cruza el nivel de compra o Bid cruza el nivel de venta, ejecuta una orden de mercado instantanea con SL (y TP si esta habilitado).

4

La posicion se gestiona sola

Una vez abierta la posicion, el Trailing Stop mueve el SL automaticamente para proteger ganancias (se activa al alcanzar breakeven). La posicion se cierra por SL, TP, trailing, o manualmente con CANCEL ALL.

Parametros de Configuracion v2.01
Parametro Default Descripcion
InpNewsTime "08:30:00" Hora del evento en formato HH:MM:SS (hora del servidor MT5, no tu hora local)
InpEntryDistPips 1000 pips Distancia de los niveles de entrada respecto al precio actual (arriba para buy, abajo para sell)
InpSLPips 10000 pips Stop Loss fijo en pips desde el precio de entrada. Amplio para aguantar el latigazo inicial de noticias
InpTPPips 10000 pips Take Profit en pips desde el precio de entrada. Solo se aplica si InpEnableTP = true
InpEnableTP true Activar/desactivar Take Profit. Con false la posicion corre sin limite de ganancia (ideal con trailing)
InpTrailingStopPips 2000 pips Distancia del trailing stop. Se activa solo al alcanzar breakeven. Valor 0 = desactivado
InpLotSize 0.10 Tamano de lote por operacion
InpMagicNumber 202601 Numero magico unico para identificar las operaciones de este EA
InpLeadTimeSeconds 10 Segundos antes del evento para calcular niveles y empezar a monitorear
Flujo Completo de Operacion

Ejemplo con NASDAQ (NAS100):

1. Configuras InpNewsTime = "08:30:00" y InpEntryDistPips = 1000
2. EA se arma automaticamente (estado ARMED), muestra countdown T-MM:SS
3. A T-10s: calcula niveles — BuyLevel en 20,100 y SellLevel en 19,900 (lineas en el chart)
4. Pasa a MONITORING — vigila el precio tick a tick
5. Sale la noticia, precio sube fuerte — Ask cruza 20,100 — EA ejecuta BUY market order con SL y TP
6. Trailing Stop activo: cuando el profit alcanza breakeven, el SL empieza a subir automaticamente
7. Cierre: por TP, SL, trailing, o boton CANCEL ALL

Sobre el Trailing Stop:

El trailing solo se activa cuando la posicion alcanza breakeven (el nuevo SL >= precio de entrada para BUY, o <= para SELL). Esto evita mover el SL antes de tiempo. Una vez activo, el SL sigue al precio protegiendo ganancias.

Sobre el spread en noticias:

En noticias el spread puede ser 3-5x mayor al normal. Usa un InpSLPips amplio para no cerrarte en el latigazo inicial. El trailing se encarga de ajustar el SL despues.

Panel del EA en el Chart
  • Titulo: "NEWS STRADDLE EA v2.01" en dorado
  • Estado: Muestra el estado actual (IDLE / ARMED / MONITORING / TRADE ACTIVE)
  • Evento: Hora configurada del evento
  • Countdown: T-MM:SS cuando esta en ARMED
  • Parametros: Resumen de Dist, SL, TP (o OFF), Trail (o OFF)
  • CANCEL ALL: Boton rojo — cierra posicion abierta y resetea el EA a IDLE
  • Boton "-" / "+": Minimizar/restaurar el panel (esquina superior derecha)

Lineas en el chart:

Cuando el EA entra en MONITORING, dibuja dos lineas horizontales: verde (nivel de compra) y roja (nivel de venta). Se eliminan cuando la orden se ejecuta o se cancela.

Combinaciones de Configuracion
  • Trailing + TP OFF: InpEnableTP=false + InpTrailingStopPips=2000. La posicion corre sin limite de ganancia; el trailing protege lo ganado. Ideal para movimientos fuertes post-noticia.
  • TP fijo + Trailing: InpEnableTP=true + InpTPPips=10000 + InpTrailingStopPips=2000. Doble proteccion: si el precio llega al TP se cierra, si se revierte el trailing te protege.
  • Solo SL (sin trailing): InpTrailingStopPips=0 + InpEnableTP=true. La posicion corre con SL y TP fijos. El cierre lo decide el broker.
Tips
  • Hora del Servidor: Verifica que InpNewsTime coincida con el reloj de MT5, no con tu hora local. Puedes ver la hora del servidor en la ventana Market Watch.
  • SL amplio: En noticias el spread se dispara y hay latigazo inicial. Con 10000 pips de SL en NAS100 tenes margen suficiente para que el trailing haga su trabajo.
  • Trailing desde breakeven: El trailing NO mueve el SL hasta que la posicion este al menos en breakeven. Esto evita cierres prematuros durante la volatilidad inicial.
  • Market Orders: A diferencia de versiones anteriores, el EA usa ordenes de mercado (no pendientes). Esto elimina rechazos por tipo de orden y funciona en todos los brokers.
  • Magic Number: Si usas multiples EAs en el mismo simbolo, asegurate de que cada uno tenga un InpMagicNumber diferente.
